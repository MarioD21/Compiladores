¿Qué es el Programa Objeto?

El programa objeto es el resultado final del proceso de compilación. Se obtiene después de generar el código objeto y realizar el proceso de enlazado con bibliotecas y módulos necesarios.

Este programa ya puede ejecutarse directamente en el sistema operativo.

Objetivos

  • Generar un programa ejecutable.
  • Integrar bibliotecas necesarias.
  • Resolver referencias externas.
  • Preparar el programa para su ejecución.
  • Crear archivos ejecutables.

¿Qué es el Enlazador (Linker)?

El enlazador es un programa que une diferentes archivos objeto y bibliotecas para formar un único programa ejecutable. 

Modulo1.obj
Modulo2.obj
Biblioteca.lib

      ↓

Programa.exe

¿Qué es el Cargador (Loader)?

El cargador es el encargado de colocar el programa ejecutable en la memoria principal para que el procesador pueda ejecutarlo. 

Programa.exe
      ↓
Memoria RAM
      ↓
CPU

Bibliotecas

Las bibliotecas contienen funciones ya desarrolladas que pueden reutilizarse en distintos programas. 

printf()
scanf()
sqrt()
pow()

Tipos de Bibliotecas

Tipo Descripción
Estática Se incorpora al ejecutable durante la compilación.
Dinámica Se carga durante la ejecución del programa.

Ventajas del Programa Objeto

  • Puede ejecutarse directamente.
  • Integra múltiples módulos.
  • Permite reutilizar bibliotecas.
  • Optimiza el uso del hardware.
  • Representa el resultado final de la compilación.

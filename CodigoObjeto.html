Sexta fase del proceso de compilación

Inicio

¿Qué es la Generación de Código Objeto?

La generación de código objeto es la fase donde el compilador transforma el código intermedio optimizado en instrucciones específicas para una arquitectura de hardware.

El resultado es un archivo objeto que posteriormente podrá convertirse en un programa ejecutable.

Objetivos

  • Traducir el código intermedio a instrucciones de máquina.
  • Asignar registros del procesador.
  • Generar instrucciones eficientes.
  • Preparar el programa para el enlazador.
  • Crear archivos objeto.

Entrada y Salida

Entrada

Código Intermedio Optimizado

Salida

Código Objeto

¿Qué es el Código Objeto?

Es un conjunto de instrucciones traducidas a lenguaje máquina o lenguaje ensamblador que la computadora puede ejecutar. 

10110000 01100001
10110001 00000010
00000001 11000001

Lenguaje Ensamblador

Antes de llegar al lenguaje máquina, muchas veces el compilador genera código ensamblador. 

MOV AX, 5
MOV BX, 10
ADD AX, BX

Asignación de Registros

El compilador debe decidir en qué registros almacenar los datos. 

T1 = A + B

R1 = A
R2 = B

ADD R1,R2

Registros del Procesador

Registro Función
AX Acumulador
BX Base
CX Contador
DX Datos

Instrucciones Básicas

Instrucción Función
LOAD Cargar dato en memoria
STORE Guardar dato
ADD Sumar
SUB Restar
MUL Multiplicar
DIV Dividir

Ejemplo de Traducción

Código Intermedio

T1 = A + B

Código Objeto

LOAD R1, A
LOAD R2, B
ADD R1, R2
STORE T1, R1

Manejo de Temporales

T1 = A + B
T2 = C + D
T3 = T1 * T2
        
LOAD R1,A
LOAD R2,B
ADD R1,R2

LOAD R3,C
LOAD R4,D
ADD R3,R4

MUL R1,R3
STORE T3,R1

Problema de Registros Limitados

Los procesadores tienen una cantidad limitada de registros. 

T1 = A + B
T2 = C + D
T3 = E + F

Si solo existen dos registros disponibles, algunos temporales deberán almacenarse temporalmente en memoria. 

STORE T1
STORE T2

Código para Condicionales

if(a>b)
{
   x=1;
}
        
CMP A,B
JG L1
JMP L2

L1:
MOV X,1

L2:

Código para Ciclos

while(x<10)
{
   x++;
}
        
L1:
CMP X,10
JGE L2

INC X
JMP L1

L2:

Archivo Objeto

El código generado se almacena en archivos objeto con extensiones como:

  • .obj
  • .o

Estos archivos aún no son ejecutables.

Ventajas

  • Genera instrucciones ejecutables.
  • Aprovecha los recursos del procesador.
  • Reduce tiempos de ejecución.
  • Permite crear programas eficientes.
  • Prepara el programa para el enlazado.
